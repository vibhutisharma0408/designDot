# HLS Global Landing Page

A responsive landing page built exactly to Figma design specifications using **Next.js** and **Tailwind CSS**.

## Project Highlights
- **Responsive Design**: fully fluid layout adapting perfectly across viewports ranging from `320px` to `1920px` while adhering to precise dimension ratios.
- **Modern Framework**: bootstrapped natively with `create-next-app` rendering functional React server-side optimization.
- **Pixel-Perfect Alignment**: layout styling precisely modeled matching the Figma grids, utilizing Tailwind Flexbox and absolute positioning properties.
- **Custom UI Graphics**: native inline scalable SVGs mapped strictly replacing heavy static assets.

## Quick Start
First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.
